## 4. Project Context & Requirements

### 4.1 Project Specification
**System:** Online grant application platform for Studio Ghibli's climate change initiatives  
**Key Product:** Teto Grant (up to €10,000 funding)  
**Deadline:** 14 April 2026

### 4.2 Core Requirements
1. Capture applicant details and funding requirements
2. Map applications to grant selection (Teto, etc.)
3. Single field for special award applications
4. Applicant data immutable after submission
5. Admin (Nausicaä Enterprises) can view, edit, and manage all applications
6. Clean, user-friendly, secure interface

### 4.3 Deliverables
- Full SDLC pipeline (development → testing → deployment)
- Prototype UI page(s) for pipeline validation
- Standard security compliance and data protection
